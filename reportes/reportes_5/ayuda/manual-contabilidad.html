MANUAL DE USUARIO.

INGRESO AL SISTEMA:

Escriba su nombre de usuario y contraseña para ingresar al sistema.

Luego oprima Aceptar para Ingresar o sino oprima Limpiar para borrar los datos de las casillas de ingreso.

El sistema le permitirá ingresar si digitó su usuario y clave correctamente, de lo contrario le mostrará un mensaje de error y le impedirá continuar.

Si no puede entrar, continue probando su clave sino  consulte a soporte técnico por problema.

PANTALLA DE BIENVENIDA:

Al ingresar al sistema, aparecerá una página de bienvenida, con un menú.

Seleccione del menú la opción que desea utilizar, para comenzar a utilizar el sistema.
